Here is the translated `CMakeLists.txt` file for the OpenMP Offload execution model:

```cmake
cmake_minimum_required(VERSION 3.21)

project(
    XSBench_OpenMP
    VERSION 1.0
    LANGUAGES CXX
)

if(NOT CMAKE_BUILD_TYPE)
  set(CMAKE_BUILD_TYPE Release)
endif()

set(CMAKE_CXX_FLAGS "-Wall -Wextra")
set(CMAKE_CXX_FLAGS_DEBUG "-g")
set(CMAKE_CXX_FLAGS_RELEASE "-O3")

set(SOURCE Main.cu io.cu Simulation.cu GridInit.cu XSutils.cu Materials.cu)

# Enable OpenMP Offloading
find_package(OpenMP REQUIRED)
if(OpenMP_CXX_FOUND)
    set(CMAKE_CXX_FLAGS "${CMAKE_CXX_FLAGS} ${OpenMP_CXX_FLAGS} -fopenmp -fopenmp-targets=nvptx-none")
endif()

add_executable(XSBench ${SOURCE})

install(TARGETS XSBench DESTINATION bin)
```

### Key Changes:
1. **Project Name**: Changed from `XSBench_CUDA` to `XSBench_OpenMP` to reflect the new execution model.
2. **Removed CUDA Language**: The `LANGUAGES CXX CUDA` line was removed since we are now using OpenMP.
3. **OpenMP Support**: Added `find_package(OpenMP REQUIRED)` to find OpenMP support and set the necessary flags for OpenMP offloading.
4. **OpenMP Flags**: Added `-fopenmp` and `-fopenmp-targets=nvptx-none` to the compiler flags to enable OpenMP offloading to NVIDIA PTX (which is relevant for OpenMP offloading).

This CMake configuration is now set up to compile the project using OpenMP offloading instead of CUDA.
