int * load_num_nucs(long n_isotopes)
{
    int max_num_nucs = 0;
    #pragma omp offload schedule(dynamic) \
                  num_tasks(12)            \
                  in (n_isotopes:             \
                      out (max_num_nucs:       \
                          alloc_if(alloc::TempArray<int>) \
                          free_temp(alloc::TempArray<int>)))

        // Material 0 is a special case (fuel). The H-M small reactor uses

        else
            max_num_nucs = 321; // HM Small is 34, H-M Large is 321

        max_num_nucs = omp_get_max_work_item(0);
    }
    return (int*)malloc(max_num_nucs * sizeof(int));
}

Note that the above code uses OpenMP Offload directives to transfer data between the host and device. The `in` clause specifies the data to be transferred from the host to the device, while the `out` clause specifies the data to be transferred from the device to the host. The `alloc_if` clause is used to allocate memory on the device if it does not already exist, and the `free_temp` clause is used to free temporary memory allocated on the device.

Also note that this code assumes that the OpenMP Offload library is installed and configured properly on your system. If you are using a different offloading mechanism (such as CUDA or HIP), you will need to modify the code accordingly.
