Note that I have made the following changes:
1. Removed CUDA specific headers and macros.
2. Replaced `__global__` with regular function declarations since OpenMP Offload does not use this keyword.
3. Removed `curand_kernel.h`, `cuda.h`, and other CUDA-specific includes as they are not needed for OpenMP Offload.
4. Removed the `CUDA_CALL` macro as it is specific to CUDA error handling.
5. Removed the `CudaCheckError` function as it is specific to CUDA error handling.
6. Replaced `#define CUDA_ERROR_CHECK` with nothing since this define was used for CUDA-specific error checking.

The code now should be compilable and run correctly using OpenMP Offload, but note that you will need to modify the rest of your codebase (e.g., kernel.cu, main.cu, init.cu) as well to use OpenMP Offload instead of CUDA.
