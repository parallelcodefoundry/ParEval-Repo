Here is the translated Makefile:
```makefile
# Makefile for OpenMP microXOR

CXX ?= g++
SM_VERSION ?= $(SM_VERSION)
CXX_OFFLOAD_FLAGS ?= -fopenmp-targets=nvptx64-nvidia-cuda -fopenmp-target-exec=omp- target-device=nvidia,cuda=$(SM_VERSION)

SRC = src
INC = include
OBJ = obj

OBJS = $(foreach file,$(wildcard $(SRC)/*.cpp),$(file:$(SRC)/%.cpp=$(OBJ)/%.o))

EXE = microXOR.exe

$(EXE): $(OBJS)
	$(CXX) -fopenmp -o $@ $^ $(CXX_OFFLOAD_FLAGS)

$(OBJ)/%.o: $(SRC)/%.cpp
	$(CXX) -c -fopenmp $< -o $@

# Cleanup
clean:
	rm -f $(EXE) $(OBJS)
```
Note that I've made the following changes:

* Replaced `NVCC` with `CXX`, and set it to a compiler like `g++`.
* Removed the `-arch=$(CUDA_ARCH)` flag, as OpenMP doesn't need this.
* Added the `-fopenmp-targets=nvptx64-nvidia-cuda -fopenmp-target-exec=omp- target-device=nvidia,cuda=$(SM_VERSION)` flags to enable CUDA offloading with OpenMP.
* Changed the rule for building the executable to use `g++` and include the `-fopenmp` flag.
* Removed the `cudaMalloc`, `cudaMemcpy`, and other CUDA-related commands, as they are not needed in an OpenMP offloaded code.

You can build the code using a command like:
```bash
make SM_VERSION=80 CXX_COMPILER=clang++
```
This will compile the code for an NVIDIA GPU with compute capability 80, using clang++ as the compiler.
