### Changes Made:
1. Removed the `__global__` qualifier since OpenMP does not use this syntax.
2. Changed the function declaration for `cellsXOR` to a regular function declaration, as OpenMP will handle the offloading to the device.
3. Included the OpenMP header `<omp.h>` instead of `<cuda_runtime.h>`. 

The implementation of the `cellsXOR` function will need to be modified accordingly in the `src/microXOR.cu` file to use OpenMP directives for offloading.
