### Key Changes:
1. **Header Guards**: Changed from `MICROXOR_CUH` to `MICROXOR_HPP` to reflect the change from CUDA to C++ header file conventions.

2. **Include Directives**: Removed `#include <cuda_runtime.h>` as it's specific to CUDA and not needed for OpenMP. Added `#include <omp.h>` for OpenMP support.

3. **Function Declaration**: Changed the `__global__` CUDA kernel function declaration to a regular C++ function declaration. The function will be defined using OpenMP pragmas in the corresponding `.cpp` file.
