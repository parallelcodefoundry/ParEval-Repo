### Key Changes:
- The `#include <cuda_runtime.h>` directive has been removed because it is specific to CUDA and not needed in OpenMP.
- The `__global__` keyword has been removed from the function declaration, as it is specific to CUDA. In OpenMP, we will use `#pragma omp target` and related pragmas to offload computation to the GPU.
- The file extension has been changed from `.cuh` to `.hpp` to reflect the C++ standard header file naming convention.
